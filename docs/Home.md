# Welcome to the naarad Wiki

naarad (नारद) is a self-hosted, lightweight email tracking pixel and link analytics service. It helps you track when and where your emails are opened and clicked.

## Key Features

- **Open Tracking**: Uses an invisible 1×1 pixel to track email opens.
- **Link Tracking**: Secure click-through redirect with open-redirect protection.
- **Rich Analytics**: Collects Geo-location, ISP/ASN, device, browser, and OS data.
- **Privacy First**: Fully self-hosted so you own all your data.
- **Multi-Node Sync**: Optional synchronization from edge nodes to a central server.
- **Security**: Timing-safe API key authentication, HSTS, CSP, and rate limiting against abuse.
